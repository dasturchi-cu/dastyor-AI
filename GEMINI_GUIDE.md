# 🔑 GEMINI API KEY OLISH QO'LLANMASI (TEKIN)

Bot ishlashi uchun **Google Gemini** kalitini olish juda oson va mutlaqo tekin.

---

## 1. Google AI Studio ga kiring
- Sayt: [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
- Google (Gmail) akkauntingiz bilan kiring.

## 2. Kalit yarating
1. Chap tomonda **"Get API key"** tugmasini bosing.
2. **"Create API key"** tugmasini bosing.
3. Agar proekt so'rasa, "Create new project" tanlang.
4. **Kalitni nusxalab oling** (Bu kalit `AIza...` bilan boshlanadi).

## 3. Botga o'rnating
`.env` faylini oching va quyidagicha yozing:

```ini
GOOGLE_API_KEY=AIzaSy...SizningKalitingiz
```

*(Eski `OPENAI_API_KEY` qatorini o'chirib tashlang)*

---
✅ **Bo'ldi!** Endi botni ishga tushirsangiz hammasi ishlaydi.
